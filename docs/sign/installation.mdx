# Installation

## Obtain Project ID

Head over to [WalletConnect Cloud](https://cloud.walletconnect.com/) to sign in or sign up. Create (or use an existing) project and copy its associated project id. We will need this in a latter step.

## Install 





<Tabs groupId="platform">
<TabItem value="js" label="JS">



:::info
For additional type packages refer to our [TypeScript Guide](quickstart).

For platform-specific instructions, refer to our [React Native](quickstart) and [Node.js](quickstart) guides.
:::


```bash
npm install --save @walletconnect/sign-client@rc
```

## Initialize Client

Initialize client by passing `projectId` we created before.

```js
import SignClient from "@walletconnect/sign-client";

const signClient = await SignClient.init({
projectId: "<YOUR_PROJECT_ID>",
});
```

<TabItem value="swift" label="Swift">

You can add a WalletConnect SDK to your project with Swift Package Manager. In order to do that:

1. Open XCode
2. Go to File -> Add Packages
3. Paste the repo GitHub url: https://github.com/WalletConnect/WalletConnectSwiftV2
4. Tap Add Package
5. Select WalletConnect check mark
